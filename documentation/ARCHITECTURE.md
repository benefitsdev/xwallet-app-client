## Data Flow

### Wallet Creation
```
User submits form
  вҶ’ ethers.Wallet.createRandom()
  вҶ’ wallet.encrypt(pbPassword)     // encrypt private key
  вҶ’ POST /api/collections/wallets/records  // store encrypted
  вҶ’ Display public address
```

### Sending ETH
```
User fills Send form
  вҶ’ Fetch encrypted_data from PocketBase
  вҶ’ wallet.decrypt(pbPassword)     // decrypt private key in-memory
  вҶ’ signer.sendTransaction({to, value})
  вҶ’ POST /api/collections/transactions/records
  вҶ’ Poll for receipt, update status
```

### Balance Fetching
```
Dashboard loads
  вҶ’ GET /api/collections/wallets/records (user's wallets)
  вҶ’ For each wallet:
      provider.getBalance(public_address)
      вҶ’ format ETH
  вҶ’ Display all balances
```

## Security Model

| Concern | Approach |
|---|---|
| Private key storage | AES-encrypted with user's PocketBase password via ethers `encrypt()` |
| Key in memory | Decrypted only during send operation, never persisted to localStorage |
| API access | PocketBase collection rules restrict reads/writes to owner |
| Admin access | PocketBase superuser role checked via `auth.role` |
| Transport | HTTPS between browser вҶ” PocketBase |
| RPC | API key stored in server env (or frontend .env for MVP) |

## PocketBase Configuration

### Collection Rules

**wallets**
- List: `@request.auth.id != ""`
- View: `@request.auth.id = user_id.id`
- Create: `@request.auth.id != ""`
- Update: `@request.auth.id = user_id.id`
- Delete: `@request.auth.id = user_id.id`

**transactions**
- List: `@request.auth.id != ""`
- View: `wallet_id.user_id.id = @request.auth.id`
- Create: `@request.auth.id != ""`
- Update/Owner only (for status updates)

**assets**
- List: public
- View: public

## Environment Variables

See root `.env` and `app/.env` for the full list.

All frontend branding is driven by `VITE_*` prefixed vars:
- `VITE_APP_NAME` вҖ” brand name (default: "ZKX Wallet")
- `VITE_PRIMARY_COLOR` вҖ” primary hue (default: indigo)
- `VITE_BACKGROUND_COLOR` вҖ” background (default: slate-900)
- `VITE_FONT_FAMILY` вҖ” typography (default: Inter)
- `VITE_LOGO_PATH` вҖ” logo URL (default: /logo.svg)
- `VITE_RPC_URL` вҖ” Ethereum RPC endpoint
- `VITE_CHAIN_ID` вҖ” network chain ID
- `VITE_EXPLORER_URL` вҖ” block explorer base URL
- `VITE_POCKETBASE_URL` вҖ” PocketBase server URL
